Coalesce Transform MCP

MCP server for the Coalesce Transform API

An MCP (Model Context Protocol) server that enables AI agents to manage Coalesce Transform workspaces and nodes — building pipelines via YAML/config rather than raw SQL.

Features

  • Workspace and node management
  • Pipeline creation and orchestration
  • Intelligent node type selection
  • Column-level transform management
  • Join condition and aggregation support
